# 학생 매뉴얼

---

## 로그인

1. 브라우저에서 서비스 주소로 접속합니다.
2. **학생** 탭을 선택합니다.
3. 담임 선생님에게 받은 **수험번호**와 **비밀번호**를 입력합니다.
4. **로그인** 버튼을 누릅니다.

> 수험번호 예시: `1001`, `1002`

---

## 학생 대시보드

로그인하면 대시보드로 이동합니다.

| 항목 | 설명 |
|------|------|
| **오늘 학습** | 오늘 학습해야 할 DAY 바로가기 버튼 |
| **전체 진도** | 60일 중 완료한 DAY 수 및 진도율 |
| **다음 시험** | 예정된 시험 정보 및 대기실 입장 버튼 |

---

## 단어 학습

### 학습 시작

1. 대시보드에서 **오늘 학습** 버튼을 누르거나, 하단 메뉴에서 학습하고 싶은 DAY를 선택합니다.
2. 해당 DAY의 50단어 학습이 시작됩니다.

### 플래시카드 사용법

- **카드 앞면**: 영어 단어가 표시됩니다.
- **카드 탭 또는 클릭**: 카드가 뒤집히며 한국어 의미와 동의어가 나타납니다.
- 카드를 확인한 후 아래 3가지 중 하나를 선택합니다:

| 버튼 | 의미 |
|------|------|
| **외웠어요** | 확실하게 암기한 단어 (초록색) |
| **다시 볼게요** | 애매하게 아는 단어 (노란색) |
| **몰라요** | 전혀 모르는 단어 (빨간색) |

### 학습 완료 후

- 50단어를 모두 체크하면 **셀프테스트** 모드로 진입합니다.
- 셀프테스트에서 다시 한번 단어를 점검할 수 있습니다.
- 완료 후 학습 기록이 저장되고 진도가 업데이트됩니다.

---

## 오답 복습

1. 하단 메뉴에서 **복습** 탭을 선택합니다.
2. **몰라요** 또는 **다시 볼게요**로 표시한 단어 목록이 나타납니다.
3. **복습 시작** 버튼을 누르면 오답 단어만 모아서 다시 학습합니다.

---

## CBT 시험

### 시험 대기실

- 담임 선생님이 시험을 시작하면 **시험 대기실** 버튼이 활성화됩니다.
- 시험 시작 전까지 대기실에서 기다립니다.
- 시험이 시작되면 자동으로 시험 화면으로 이동합니다.

### 시험 응시

- **총 50문항**, 2개 DAY에서 각 25문항씩 출제됩니다.
- **문제 형식**: 영어 단어를 보고 **한국어 의미**를 직접 입력합니다.
- 화면 상단에 **남은 시간**이 표시됩니다.
- 문항 번호를 클릭하면 해당 문항으로 바로 이동합니다.

> **주의**: 시간이 종료되거나 담임 선생님이 시험을 종료하면 자동으로 제출됩니다.

### 시험 제출

- 모든 문항 입력 후 **제출** 버튼을 누릅니다.
- 제출 전 확인 메시지가 표시됩니다.

### 시험 결과 확인

제출 후 결과 화면에서 다음을 확인할 수 있습니다:

- 최종 점수 (50점 만점)
- 정답·오답 여부
- **오답 리뷰**: 틀린 문항의 정답 확인

---

## 상담 신청 (학생)

### 신규 신청

1. 하단 메뉴에서 **상담** 탭을 선택합니다.
2. 담임 선생님의 상담 가능 날짜와 시간이 달력으로 표시됩니다.
3. 원하는 **날짜**를 선택하면 그날의 가능한 시간대가 나타납니다.
4. **시간대**를 선택하고, 상담 요청 내용을 간략히 입력합니다.
5. **신청하기** 버튼을 누릅니다.

### 신청 수정·취소

- 이미 신청한 상담이 **예약 상태**인 경우, 날짜와 시간을 변경할 수 있습니다.
- **취소하기** 버튼을 누르면 상담 신청이 취소됩니다.
- 확정된 상담은 담임 선생님께 직접 연락하여 변경 요청합니다.

# 관리자 매뉴얼

---

## 관리자 로그인

1. 브라우저에서 서비스 주소로 접속합니다.
2. **관리자** 탭을 선택합니다.
3. 이메일 주소와 비밀번호를 입력합니다.
4. **로그인** 버튼을 누릅니다.

> 역할에 따라 접근 가능한 메뉴가 다릅니다.
> - **전체관리자**: 모든 메뉴 접근 가능
> - **지점관리자**: 담당 지점의 반·학생·시험·상담
> - **반담임**: 담당 반의 학생·시험·상담 + 상담 상세 관리

---

## 관리자 대시보드

로그인 후 자동으로 대시보드로 이동합니다.

### 요약 카드

| 카드 | 내용 |
|------|------|
| **총 학생** | 담당 범위 내 전체 학생 수 |
| **오늘 학습** | 오늘 학습 활동이 있는 학생 수 |
| **위험 학생** | 최근 3일 연속 미학습 학생 수 |
| **상담 신청** | 현재 예약(scheduled) 상태의 상담 건수 |

### 진행 중인 시험

현재 active 상태의 시험이 있으면 빨간색 배너로 표시됩니다.  
**관리** 버튼을 눌러 바로 시험 관리 화면으로 이동할 수 있습니다.

### 위험 학생 목록

3일 이상 학습 기록이 없는 학생 목록입니다.  
학생 이름을 클릭하면 해당 학생의 상세 페이지로 이동합니다.

### 당일 상담 예약

오늘 날짜에 예약된 상담 전체 목록입니다 (예약·확정·완료·취소 모두 포함).  
**상세 →** 링크를 클릭하면 상담 상세 화면으로 이동합니다.

### AI 학급 인사이트

- 날짜가 바뀌면 대시보드 접속 시 **자동으로 인사이트를 생성**합니다.
- 생성 중에는 로딩 애니메이션이 표시됩니다.
- **재생성** 버튼으로 수동으로 다시 생성할 수 있습니다.
- 학급 학습 현황, 주의 학생, 제안 사항 등이 포함됩니다.

---

## 지점 관리

> **전체관리자**만 접근 가능합니다.

### 지점 추가

1. 좌측 메뉴에서 **지점 관리**를 선택합니다.
2. **지점 추가** 입력란에 지점 이름을 입력합니다.
3. **추가** 버튼을 누릅니다.

### 지점 수정

- 지점 이름 옆 **수정** 아이콘을 클릭하면 인라인 수정이 가능합니다.
- 수정 후 **저장** 버튼을 누릅니다.

### 지점 삭제

- **삭제** 버튼을 누르면 확인 메시지가 표시됩니다.
- 확인 후 삭제됩니다.

> **주의**: 지점을 삭제하면 해당 지점의 반과 학생 데이터도 함께 삭제됩니다.

---

## 반 관리

> **전체관리자**, **지점관리자** 접근 가능합니다.

### 반 추가

1. **반 관리** 메뉴로 이동합니다.
2. 지점을 선택합니다 (전체관리자의 경우).
3. **반 이름**, **시작일**, **종료일**(시작일 + 60일)을 입력합니다.
4. **추가** 버튼을 누릅니다.

### 반 삭제

- **삭제** 버튼 클릭 후 확인 메시지에서 승인합니다.

> **주의**: 반을 삭제하면 해당 반의 학생·시험·상담 데이터도 함께 삭제됩니다.

---

## 관리자 계정 관리

> **전체관리자**만 접근 가능합니다.

### 관리자 초대 (계정 생성)

1. **관리자 관리** 메뉴로 이동합니다.
2. **관리자 추가** 폼에서 다음 항목을 입력합니다:
   - 이름
   - 이메일
   - 초기 비밀번호
   - 역할 (지점관리자 / 반담임)
   - 담당 지점 (지점관리자인 경우)
   - 담당 반 (반담임인 경우)
3. **추가** 버튼을 누릅니다.

### 비활성화 / 활성화

- 관리자 목록에서 **비활성화** 버튼을 클릭하면 해당 계정이 로그인 불가 상태가 됩니다.
- **활성화** 버튼으로 복구할 수 있습니다.
- 각 버튼 클릭 시 확인 메시지가 표시됩니다.

---

## 학생 관리

> **전체관리자**, **지점관리자**, **반담임** 모두 접근 가능합니다. (담당 범위 내)

### 학생 추가

1. **학생 관리** 메뉴로 이동합니다.
2. **학생 추가** 버튼을 클릭합니다.
3. 다음 항목을 입력합니다:
   - 이름
   - 수험번호 (로그인 ID로 사용)
   - 초기 비밀번호
   - 소속 반
4. **추가** 버튼을 누릅니다.

### 학생 상세 확인

- 학생 이름을 클릭하면 상세 페이지로 이동합니다.
- **학습 히트맵**: GitHub 스타일로 일별 학습 현황 시각화
- **시험 점수 추이**: 응시한 시험의 점수 변화 차트
- **학습 현황**: DAY별 진도 및 오답 단어 수

## 시험 출제 및 관리

> **반담임**이 주로 사용합니다.

### 시험 출제

1. **시험 관리** 메뉴로 이동합니다.
2. **시험 출제** 버튼을 클릭합니다.
3. 다음 항목을 설정합니다:

| 항목 | 설명 |
|------|------|
| 시험 제목 | 예: "2주차 단어 시험" |
| 범위 DAY 1 | 첫 번째 DAY (25문항 출제) |
| 범위 DAY 2 | 두 번째 DAY (25문항 출제) |
| 시험 시작 일시 | 날짜와 시간 선택 |
| 시험 시간 | 분 단위 (기본 8분) |

4. **출제** 버튼을 누릅니다.
5. 문항은 기출 횟수가 낮은 단어 우선으로 자동 생성됩니다.

### 시험 시작

- 시험 목록에서 예약된 시험을 선택합니다.
- **시험 시작** 버튼을 클릭하면 확인 후 학생들에게 시험이 열립니다.

### 시험 진행 중 관리

- **실시간 제출 현황**: 제출한 학생 수 확인
- **강제 종료**: 모든 학생의 답안을 강제 제출 처리합니다.

### 시험 종료 및 결과 확인

- 시험 시간이 종료되면 자동으로 `closed` 상태로 변경됩니다.
- **결과 보기** 탭에서 학생별 점수와 정오답을 확인합니다.

---

## 어휘 관리

> **전체관리자**만 접근 가능합니다.

### 어휘 목록 조회

1. **어휘 관리** 메뉴로 이동합니다.
2. DAY 필터를 선택하면 해당 DAY의 50단어 목록이 표시됩니다.

### 단어 의미·동의어 편집

1. 단어 행을 클릭하거나 **편집** 버튼을 누릅니다.
2. 편집 모달에서 다음을 수정할 수 있습니다:
   - 품사별 한국어 의미 추가·수정·삭제
   - 영어 동의어 추가·삭제
   - 기출 횟수 리셋
3. **저장** 버튼을 누릅니다.

### AI 의미 일괄 생성

- **AI 의미 일괄 생성** 버튼을 클릭합니다.
- 의미가 없는 단어들을 Google Gemini AI가 자동으로 한국어 의미를 생성합니다.
- 완료 후 각 단어에 의미가 저장됩니다.

> **주의**: AI 생성은 시간이 걸릴 수 있습니다. 완료까지 기다려 주세요.

## 상담 관리

> **반담임**이 주로 사용합니다. (상담 상세는 반담임 전용)

### 상담 현황 통계

상담 관리 페이지 상단에 현황 카드가 표시됩니다:

| 카드 | 내용 |
|------|------|
| **예약** | 학생이 신청 후 일정 미확정 상태 |
| **확정** | 담임이 일정을 확인·확정한 상태 |
| **완료** | 상담이 진행되고 기록된 상태 |
| **취소** | 취소 또는 기각된 상담 |

### 상담 신청 목록 탭

- **전체 / 예약 / 확정 / 완료 / 취소** 탭으로 상태별 필터링
- 학생 이름, 수험번호, 신청 경로(직접신청/AI추천), 날짜 확인
- **상세 보기** 버튼으로 상담 상세 페이지 이동

### 상담 시간대 탭

담임이 상담 가능한 시간대를 등록합니다.

**개별 슬롯 등록:**
1. 달력에서 날짜를 선택합니다.
2. 가능한 시간대 (9시~16시 중 선택)를 체크합니다.
3. **저장** 버튼을 누릅니다.

**일괄 등록:**
1. 시작 날짜와 종료 날짜를 선택합니다.
2. 반복할 요일과 시간대를 선택합니다.
3. **일괄 등록** 버튼을 누릅니다.

**슬롯 삭제:**
- 등록된 슬롯의 **×** 버튼을 누르면 삭제됩니다.

### 상담 상세 화면

> 반담임만 접근 가능합니다.

**상담 정보 확인:**
- 학생 이름, 수험번호, 소속 반
- 신청 경로 및 요청 내용
- AI 위험도 점수 (AI 추천인 경우)
- 이전 상담 이력 최대 5건

**일정 배정:**
1. **일정 변경** 버튼을 클릭합니다.
2. 달력에서 날짜를 선택하면 가능한 시간대가 표시됩니다.
3. 시간대를 선택합니다.
4. **예약 저장** 버튼을 클릭합니다. (확인 후 저장)

**상태 변경:**

| 버튼 | 조건 | 설명 |
|------|------|------|
| **확정** | 예약 상태 | 일정 확정 처리 |
| **완료** | 확정 상태 | 상담 완료 처리 |
| **취소** | 예약·확정 상태 | 상담 취소 처리 |

> 모든 상태 변경 시 확인 메시지가 표시됩니다.

**상담 기록 작성:**
- 상담 완료 후 **상담 내용**과 **결과** (정상복귀 / 집중관리 / 기타)를 입력합니다.
- **기록 저장** 버튼을 누릅니다.
