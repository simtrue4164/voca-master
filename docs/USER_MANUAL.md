# 학생 매뉴얼

---

## 로그인

1. 브라우저에서 서비스 주소로 접속합니다.
2. **학생** 탭을 선택합니다.
3. 담임 선생님에게 받은 **수험번호**와 **비밀번호**를 입력합니다.
4. **로그인** 버튼을 누릅니다.

> 수험번호 예시: `1007`

---

## 학생 대시보드

로그인하면 대시보드로 이동합니다.

| 항목 | 설명 |
|------|------|
| **AI 성과 예측** | 화면 최상단 — 다음 시험 예상 점수 범위, 완주 가능성, 실천 행동 제안 (당일 1회 자동 생성) |
| **오늘 학습** | 오늘 학습해야 할 DAY 바로가기 버튼 |
| **전체 진도** | 60일 중 완료한 DAY 수 및 진도율 |
| **다음 시험** | 예정된 시험 정보 및 대기실 입장 버튼 |

---

## 단어 학습

### 학습 시작

1. 대시보드에서 **오늘 학습** 버튼을 누르거나 학습하고 싶은 DAY를 선택합니다.
2. 해당 DAY의 50단어 플래시카드 학습이 시작됩니다.

### 플래시카드 사용법

- **카드 앞면**: 영어 단어가 표시됩니다.
- **카드 탭 또는 클릭**: 카드가 뒤집히며 한국어 의미와 동의어가 나타납니다.
- 카드를 확인한 후 아래 3가지 중 하나를 선택합니다:

| 버튼 | 의미 |
|------|------|
| **외웠어요** | 확실하게 암기한 단어 |
| **다시 볼게요** | 애매하게 아는 단어 |
| **몰라요** | 전혀 모르는 단어 |

### 학습 완료 후

- 50단어를 모두 체크하면 **완료 화면**이 나타납니다.
- **관련단어 학습하기** 버튼을 눌러 동의어·유의어·반의어 학습으로 이어갈 수 있습니다.

---

## 관련단어 학습

어휘 학습 완료 후 해당 DAY의 동의어·유의어·반의어를 카드로 학습합니다.

### 사용법

- 카드 앞면: 관련단어 + 타입 뱃지 (동의어 / 유의어 / 반의어)
- 카드 뒤집기: 원본 어휘 + 한국어 의미 확인
- 알겠어요 / 모르겠어요 버튼으로 학습 상태를 기록합니다.

### 학습 진도 추적

- 동의어·유의어·반의어 각각의 학습률이 독립적으로 집계됩니다.
- 관리자 학습 진도 화면에서 DAY별로 확인 가능합니다.

---

## 오답 복습

1. **복습** 탭을 선택합니다.
2. **몰라요**로 표시한 단어 목록이 나타납니다.
3. **복습 시작** 버튼을 누르면 오답 단어만 모아서 다시 학습합니다.

---

## CBT 시험

### 시험 응시

- 담임 선생님이 시험을 시작하면 시험 화면이 활성화됩니다.
- **총 50문항**, 2개 DAY에서 각 25문항씩 출제됩니다.
- **문제 형식**: 영어 단어를 보고 **한국어 의미**를 직접 입력합니다.
- 화면 상단에 **남은 시간**이 표시됩니다.

> **주의**: 시간이 종료되거나 담임 선생님이 시험을 종료하면 자동으로 제출됩니다.

### 시험 결과 확인

제출 후 결과 화면에서 확인할 수 있습니다:
- 최종 점수 (50점 만점)
- 정답·오답 여부
- 틀린 문항의 정답 확인
- **AI 코칭 메시지**: 점수 카드 바로 아래에 점수·오답 단어·학습 진도율 기반 맞춤 조언이 자동 표시됩니다.

---

## 상담 신청

1. **상담** 탭을 선택합니다.
2. 담임 선생님의 상담 가능 날짜와 시간이 달력으로 표시됩니다.
3. **날짜** 선택 → **시간대** 선택 → 상담 요청 내용 입력 → **신청하기**

- 예약 상태인 경우 날짜·시간 변경이 가능합니다.
- **취소하기** 버튼으로 상담 신청을 취소할 수 있습니다.

# 관리자 매뉴얼

---

## 관리자 로그인

1. 브라우저에서 서비스 주소로 접속합니다.
2. **관리자** 탭을 선택합니다.
3. 이메일 주소와 비밀번호를 입력합니다.

> 역할에 따라 접근 가능한 메뉴가 다릅니다.
> - **전체관리자**: 모든 메뉴 접근 가능
> - **지점관리자**: 담당 지점의 반·학생·시험·상담
> - **반담임**: 담당 반의 학생·학습진도·시험·상담

---

## 학년도 선택

화면 우측 상단 헤더에 **학년도 드롭다운**이 표시됩니다.

- 학년도를 선택하면 반관리·학생관리·학습진도·시험관리 전체 메뉴에 반영됩니다.
- 선택한 학년도는 브라우저에 저장되어 다음 접속 시에도 유지됩니다.
- 각 페이지에서 **지점 → 학년도 → 반** 순서로 세부 필터를 추가로 적용할 수 있습니다.

---

## 관리자 대시보드

### 요약 카드

| 카드 | 내용 |
|------|------|
| **총 학생** | 담당 범위 내 전체 학생 수 |
| **오늘 학습** | 오늘 학습 활동이 있는 학생 수 |
| **위험 학생** | 최근 3일 연속 미학습 학생 수 |
| **상담 신청** | 예약(scheduled) + 확정(confirmed) 상태의 상담 건수 |

### 위험 학생 목록

3일 이상 학습 기록이 없는 학생 목록입니다.  
학생 이름을 클릭하면 해당 학생의 상세 페이지로 이동합니다.

### 당일 상담 예약

오늘 날짜에 예약된 상담 전체 목록입니다.  
**상세 →** 링크를 클릭하면 상담 상세 화면으로 이동합니다.

### AI 학급 인사이트

- 날짜가 바뀌면 대시보드 접속 시 **자동으로 생성**됩니다.
- **재생성** 버튼으로 수동 재생성이 가능합니다.

---

## 지점 관리

> **전체관리자**만 접근 가능합니다.

- **지점 추가**: 지점 이름 입력 후 추가 버튼
- **수정**: 지점 이름 옆 수정 아이콘 클릭 → 인라인 편집
- **삭제**: 삭제 버튼 → 확인 후 삭제

> 지점 삭제 시 해당 지점의 반·학생 데이터도 함께 삭제됩니다.

---

## 반 관리

> **전체관리자**, **지점관리자** 접근 가능합니다.

### 검색 필터

- **지점** → **학년도** 순서로 필터를 선택하면 해당 조건의 반 목록이 조회됩니다.
- 비활성화된 반은 검색 결과에서 제외됩니다.

### 반 추가

지점, 학년도, 반 이름, 시작일을 입력한 후 **추가** 버튼을 누릅니다.

### 담임 배정

반 목록에서 담임 드롭다운을 선택하면 즉시 저장됩니다.

### 반 수정 및 비활성화

- **수정** 버튼: 반 이름, 학년도, 시작일 변경 가능
- **활성/비활성 토글**: 비활성화된 반은 학생 관리·시험 등 모든 필터에서 제외됩니다.

### 반 삭제

소속 학생이 없는 반만 삭제 가능합니다.

---

## 관리자 계정 관리

> **전체관리자**만 접근 가능합니다.

### 관리자 초대

이름, 이메일, 초기 비밀번호, 역할, 담당 지점/반을 입력한 후 **추가** 버튼을 누릅니다.

### 비활성화 / 활성화

관리자 목록에서 **비활성화** 버튼 클릭 시 해당 계정의 로그인이 차단됩니다.  
**활성화** 버튼으로 복구할 수 있습니다.

---

## 학생 관리

> **전체관리자**, **지점관리자**, **반담임** 모두 접근 가능합니다. (담당 범위 내)

### 검색 필터

페이지 상단의 **지점 → 학년도 → 반** 필터로 원하는 범위의 학생만 조회합니다.

### 학생 추가

이름, 수험번호, 초기 비밀번호, 소속 반을 입력한 후 **추가** 버튼을 누릅니다.

### 월별 배정 체크박스

학생 목록의 1~12월 체크박스로 해당 학생의 월별 수강 여부를 관리합니다.
- 체크: 해당 월에 현재 소속 반으로 배정
- 헤더 체크박스: 전체 학생 일괄 토글

### 학생 수정

**수정** 버튼 클릭 시 이름, 계정 상태, 반 배정, 비밀번호 초기화가 가능합니다.

### 학생 상세 확인

학생 이름을 클릭하면 상세 페이지로 이동합니다.
- 학습 히트맵
- 어휘 진도 및 관련단어 진도 (DAY별)

---

## 학습 진도 관리

> **전체관리자**, **지점관리자**, **반담임** 모두 접근 가능합니다.

### 검색 필터

**지점 → 학년도 → 반** 순서로 필터를 적용합니다.

### 반별 요약 카드

필터 조건에 해당하는 반마다 카드가 표시됩니다.
- 어휘 평균 진도율 (파란색 막대)
- 관련단어 평균 진도율 (보라색 막대)
- 위험 학생 수 (3일 이상 미학습)

### 학생별 진도 테이블

| 컬럼 | 설명 |
|------|------|
| **상태 점** | 초록(오늘 학습) / 노랑(1~2일 전) / 빨강(3일+ 미학습) |
| **어휘 진도** | 전체 3,000단어 대비 학습 완료 비율 |
| **관련 단어** | 동의어·유의어·반의어 전체 대비 학습 비율 |
| **최근 학습** | 마지막 학습 날짜 |

### 학생별 상세 진도

**상세** 버튼을 클릭하면 해당 학생의 상세 진도 페이지로 이동합니다.

- **요약 카드**: 어휘 진도%, 어휘 학습율%, 관련단어%, 현재 Day
- **어휘 학습 진도**: 전체 진도 바 + DAY별 미니 블록 (파란색 = 해당 DAY 50단어 완료)
- **관련단어 현황**: 동의어·유의어·반의어 타입별 요약 + DAY별 상세 테이블

## 시험 출제 및 관리

### 검색 필터

페이지 상단의 **지점 → 학년도 → 반** 필터로 원하는 범위의 시험만 조회합니다.

### 시험 출제

| 항목 | 설명 |
|------|------|
| 시험 제목 | 예: "2주차 단어 시험" |
| 범위 DAY 1 | 첫 번째 DAY (25문항) |
| 범위 DAY 2 | 두 번째 DAY (25문항) |
| 시험 시작 일시 | 날짜와 시간 선택 |
| 시험 시간 | 분 단위 (기본 8분) |

**출제** 버튼을 누르면 기출 횟수가 낮은 단어 우선으로 50문항이 자동 생성됩니다.

### 시험 시작 / 종료

- **시험 시작** 클릭 → 확인 → 학생들에게 시험 오픈
- **강제 종료** 클릭 → 전체 학생 답안 강제 제출 처리

### 시험 결과 확인

종료된 시험의 **결과 보기**에서 학생별 점수와 정오답을 확인합니다.

---

## 어휘 관리

> **전체관리자**만 접근 가능합니다.

### 단어 의미·관련단어 편집

단어 행의 **편집** 버튼을 클릭하면 편집 모달이 열립니다.
- 품사별 한국어 의미 추가·수정·삭제
- 동의어·유의어·반의어 추가·삭제
- 기출 횟수 리셋

### AI 의미 일괄 생성

**AI 의미 일괄 생성** 버튼을 클릭하면 의미가 없는 단어들의 한국어 의미, 동의어, 유의어, 반의어를 Gemini AI가 자동으로 생성합니다.

> 3,000단어 전체 생성 시 시간이 걸릴 수 있습니다.

---

## 상담 관리

> 상담 목록/통계는 전체 역할 접근 가능, **상담 상세는 반담임 전용**입니다.

### 상담 현황 통계

| 상태 | 설명 |
|------|------|
| **예약** (scheduled) | 학생 신청 또는 슬롯 배정 완료, 일정 미확정 단계 |
| **확정** (confirmed) | 담임이 "확정" 버튼으로 일정 최종 확인 |
| **완료** (completed) | 상담 진행 후 기록 저장 완료 |
| **취소** (dismissed/cancelled) | 취소 또는 기각 |

### 상담 신청 목록

상태별 탭으로 필터링하여 학생 이름, 신청 경로(직접신청/AI추천), 날짜를 확인합니다.

### 상담 시간대 등록

1. 달력에서 날짜를 선택합니다.
2. 가능한 시간대(9시~16시)를 체크합니다.
3. **저장** 버튼을 누릅니다.

### 상담 상세 (반담임 전용)

- **일정 배정**: 달력에서 날짜/시간 선택 → **예약변경** 버튼으로 슬롯 지정 (예약 상태에서 가능)
- **확정**: 예약 상태에서 **확정** 버튼 클릭 → confirmed 상태로 전환
- **완료 처리**: 확정 상태에서 상담 내용 + 결과(정상복귀/집중관리/기타) 입력 → **완료 저장**
- **취소**: 각 단계에서 **취소** 버튼 → 확인 모달 후 dismissed 상태로 전환
- 모든 상태 변경은 native confirm 창 대신 **커스텀 확인 모달**로 처리됩니다.
