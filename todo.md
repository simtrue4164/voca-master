# 📘 [Project] Voca-Master: 편입영어 AI 맞춤형 튜터
> **AI 기반 고난도 어휘 추출 및 루브릭 기반 형성적 피드백 시스템**

## 1. 개요 (Introduction)
* **대상:** 고난도 어휘 습득과 문맥적 이해가 필수적인 **편입 영어 수험생**
* **핵심 가치:** 단순 암기를 넘어, AI를 통해 단어의 **정확한 용례와 동의어를 학습**하고 실시간 **문장 작성 피드백**을 통해 실질적인 언어 구사 능력을 배양함.

---

## 2. 주요 기능 (Core Features)

### 🔍 스마트 단어 조회 및 확장
* 사용자가 단어만 입력하면 AI가 실시간으로 데이터를 생성합니다.
* **조회 항목:** 품사, 한국어 의미, **편입 빈출 동의어(Synonyms)**, 예문.
* **효율성:** 별도의 DB 구축 없이 LLM API를 통해 방대한 영단어 데이터를 즉시 활용합니다.

### ✍️ 루브릭 기반 실시간 문장 첨삭 (Formative Feedback)
* 학습한 단어를 활용해 사용자가 예문을 작성하면 AI가 즉시 교정합니다.
* **평가 루브릭:**
    * **Grammar:** 문법적 오류 확인
    * **Context:** 단어의 의미적/문맥적 적절성
    * **Complexity:** 편입 수준에 걸맞은 문장 구조 권장

### 📈 학습 성과 관리 대시보드
* 조회한 단어와 퀴즈 수행 결과를 저장하여 개인별 취약 단어를 분석합니다.
* **Mastery Level:** 복습 횟수와 첨삭 통과 여부에 따른 숙련도 시각화.

---

## 3. 기술 스택 (Tech Stack)

| 구분 | 기술 스택 | 비고 |
| :--- | :--- | :--- |
| **Frontend** | React.js / Tailwind CSS | 반응형 웹 디자인 (모바일 학습 최적화) |
| **Backend** | Python (FastAPI / Flask) | AI API 연동 및 비즈니스 로직 처리 |
| **Database** | MS-SQL | 학습 이력 및 단어 캐싱 데이터 관리 |
| **AI Model** | GPT-4o / Gemini 1.5 Pro | 단어 정보 생성 및 루브릭 기반 첨삭 에이전트 |

---

## 4. 시스템 아키텍처 (System Architecture)

1.  **User Input:** 사용자가 단어 입력 (예: `prevalent`)
2.  **AI Orchestration:** Backend에서 AI API 호출 (Prompt Engineering 적용)
3.  **Data Processing:** * AI가 생성한 JSON 데이터를 파싱하여 화면에 출력
    * 동시에 MS-SQL에 저장 (동일 단어 재요청 시 성능 최적화)
4.  **Feedback Loop:** 사용자의 문장 작성 -> AI 루브릭 평가 -> 결과 리포트 제공

---

## 5. AI 협업 및 효율성 (AI Collaboration)

본 프로젝트는 **AI와의 효율적 협업**을 심사 기준으로 삼아 다음과 같은 과정을 거쳤습니다.

* **프롬프트 엔지니어링:** 단어의 단순 의미를 넘어 '편입 영어'라는 특수 목적에 맞는 동의어를 추출하도록 페르소나 설정.
* **데이터 구축 효율화:** 수천 개의 단어 데이터를 직접 입력하는 대신, LLM을 동적 데이터베이스로 활용하여 개발 기간 단축.
* **코드 리팩토링:** 복잡한 SQL 프로시저와 Python API 연동 로직 설계 시 AI를 활용하여 코드의 안정성 확보.

## 6. 기대 효과 (Expected Effects)
* **학습 시간 단축:** 모르는 단어의 동의어와 예문을 찾는 물리적 시간 대폭 감소.
* **정교한 학습:** 전문 강사의 첨삭 없이도 AI를 통해 24시간 실시간 피드백 수신 가능.
* **실무 접합성:** 실제 교육 현장에서 발생하는 '채점 및 피드백 지연' 문제를 기술적으로 해결.
